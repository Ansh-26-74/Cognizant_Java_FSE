# Digital Nurture 4.0 – Java FSE

This repository contains my solutions to the exercises and assignments completed as part of the **Digital Nurture 4.0 – Java Full Stack Engineer (FSE)** program.

The repository covers core Java concepts, Design Patterns, Data Structures & Algorithms (DSA), PL/SQL, and other programming exercises. Each exercise is organized into its own project with source code and supporting documentation.

## Repository Structure

### Design Patterns

* Exercise 1 – Singleton Pattern
* Exercise 2 – Factory Method Pattern

### Data Structures & Algorithms (DSA)

* Exercise 1 – Inventory Management System
* Exercise 2 – E-commerce Platform Search Function
* Exercise 7 – Financial Forecasting

### PL/SQL

* Exercise 1 – Control Structures

    * Scenario 1 – Apply a 1% discount to loan interest rates for customers above 60 years.
    * Scenario 2 – Promote customers to VIP status based on account balance.
    * Scenario 3 – Generate reminders for customers with loans due within the next 30 days.

> More exercises and projects will be added as I progress through the Digital Nurture 4.0 – Java FSE program.
