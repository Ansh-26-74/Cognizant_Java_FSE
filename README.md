# Digital Nurture 4.0 – Java Full Stack Engineer (FSE)

This repository contains my solutions to the exercises, assignments, and hands-on projects completed as part of the **Digital Nurture 4.0 – Java Full Stack Engineer (FSE)** program by Cognizant.

The repository showcases practical implementations of **Core Java, Spring Framework, Design Patterns, Data Structures & Algorithms (DSA), PL/SQL, SQL, Maven, and Object-Oriented Programming (OOP)** concepts. Each exercise is organized into its own project with clean source code and relevant documentation.

## PL/SQL

### Exercise 1 – Control Structures

* Scenario 1 – Apply a 1% discount to loan interest rates for customers above 60 years.
* Scenario 2 – Promote customers to VIP status based on account balance.
* Scenario 3 – Generate reminders for customers with loans due within the next 30 days.

## Spring Framework

### Spring Core

#### Exercise 1 – Configuring a Basic Spring Application

* Created a Maven-based Spring Core project.
* Configured the Spring IoC Container using `applicationContext.xml`.
* Defined and managed Spring beans using XML configuration.
* Loaded and tested the Spring Application Context.

#### Exercise 2 – Implementing Dependency Injection

* Implemented Setter-Based Dependency Injection.
* Wired `BookRepository` into `BookService` using Spring IoC.
* Demonstrated loose coupling through Dependency Injection.
* Verified bean creation and dependency injection using the Spring container.

#### Exercise 4 – Creating and Configuring a Maven Project

* Created a Maven project for a Spring application.
* Added Spring Context, Spring AOP, and Spring Web MVC dependencies.
* Configured the Maven Compiler Plugin for Java 1.8 compatibility.
* Learned Maven project structure, dependency management, and build configuration.
